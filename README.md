# 🛒 Flipkart Sales Analytics Dashboard

An interactive Power BI dashboard designed to analyze Flipkart sales performance, customer purchasing behavior, payment trends, and top-selling products.  
This project helps in understanding business insights through clean visualizations and KPI analysis.

---

# 📌 Project Overview

The main objective of this project is to transform raw Flipkart sales data into meaningful business insights using Power BI.  
The dashboard provides a complete overview of sales trends, customer activity, category performance, and payment preferences.

---

# ✨ Key Features

✔️ Total Sales Analysis  
✔️ Total Orders Tracking  
✔️ Monthly Sales Trend  
✔️ Category-wise Sales Performance  
✔️ Payment Method Analysis  
✔️ Top Selling Products  
✔️ Customer Rating Insights  
✔️ Interactive KPI Cards & Visualizations

---
